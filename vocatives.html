Vocative


I don't know if there's already something like that, usually people use names


In my case a simple name is unsuitable and I wanted to shorten clarification


What do I mean by "Vocatives"


A pair of names (or set of names) similar to pronouns' `subjective/objective`


But I've decided to state vocatives as `direct/non-direct`


So if one has a name then it is `[name]/[name]` as vocatives


If one has two names (example: informal/formal) they can set `Alex/Alexander` (Alex(ander) as an example)


What does `none/any` mean?


So basically what it says is that I don't care what others refer to me as (`any`)


But I prefer not being called ANY name when talked to directly (`none`)
